Is this portal for me?

The installation of this portal is not for novices. I recommend you have experience working with web technology. If you have no experience working with databases and/or HTML then you may find the installation frustrating.

Requirements

You must use version 0.9.10 of the STHS v2 or higher with the STHS v2 Web Portal.

If you are upgrading an old STHS v1 Web Portal Classic to the new STHS v2 Web Portal, it MUST have all the SQL database upgrades installed from your initial installation version up to 2.5. If you are missing one upgrade, the upgrade process could fail and cause data corruption. So make backups of your database.

Web Hosting Requirements

  • Own one copy of STHS v2
  • 100mb of space. Web hosting can be purchased as low as $3.99 per month through Byet Hosting.
  • PHP 4 or greater
  • MySQL 4.1 or greater database server or greater

Whats included

  • PHP pages to render the website.
  • MySQL database script to build the database.
